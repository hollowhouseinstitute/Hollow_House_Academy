# Hollow House Academy

A governance-first, child-safe educational framework demonstrating how
Human Authority + Governed AI can create joyful, ethical, decolonized learning.

## Purpose
This repository is designed so that **any organization** can run Hollow House Academy
without the original author present — safely, audibly, and ethically.

## Core Principles
- Time turns behavior into infrastructure.
- Behavior is the most honest data there is.
- Human authority is never bypassed.
